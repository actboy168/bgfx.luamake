# bgfx.luamake

Build [bgfx](https://github.com/bkaradzic/bgfx) with [luamake](https://github.com/actboy168/luamake).

### Use

Build all
> luamake

Run example
> luamake -runtest 00

Or
> luamake -runtest 00-helloworld
